First Time Git and Github Setup
=======================
1. git init
2. git add .
3. git commit -m "any message"
4. git remote add origin https://github.com/karlh3/property-management.git
5. git remote set-url origin https://karlh3:MYTOKEN@github.com/karlh3/property-management.git
6. git push --set-upstream origin master

Next Time Onwards
======================
1. git pull
2. git add .
3. git commit -m "some message"
4. git push

For intellij, add token to github settings
